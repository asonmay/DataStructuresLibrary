I made this during a data strutures class taken at Great Minds Robotics during middle school.
Here is a list of all the data structures:
- Sorting Algorithms

  - Selection Sort
  - Insertion Sort
  - Quick Sort
  - Bubble Sort
- Linked List
- Stack
- Queue
- Graph
